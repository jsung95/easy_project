## 개발 목적
1. C#, WPF 및 Oracle DataBase 집중적 활용
2. 의료IT 개발자로서의 성장

## 프로그램 기능
1. 재고 현황 및 상태 체크 (표 & 차트)
2. 재고 입고/출고 내역 확인
3. 신규 재고등록, 재고수정, 발주 신청서 작성
4. 관리자라면 권한 로그 및 로그데이터 확인 가능
